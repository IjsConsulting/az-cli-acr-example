1. Clone code from gitHub. 

git clone https://github.com/Azure-Samples/aci-helloworld.git

2.Build docker image (must have docker client runing locally). 

[BEFORE]
C:\Working\....\>docker images
REPOSITORY          TAG                 IMAGE ID            CREATED             SIZE

docker build ./acr-build-helloworld-node -t aci-tutorial-app

3. View image has been created

docker images

[AFTER]
C:\Working\....\>docker images
REPOSITORY          TAG                 IMAGE ID            CREATED             SIZE
aci-tutorial-app    latest              bb65ee7d0981        15 seconds ago      68.5MB
node                9-alpine            a56170f59699        23 months ago       68.5MB

4. Run the image localy

docker run -d -p 8080:80 aci-tutorial-app

5. Navigate to Url to view application.

http://localhost:8080

The site should be running, the browser should show the text .. 

Hello World
